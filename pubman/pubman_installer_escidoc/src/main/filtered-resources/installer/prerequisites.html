PubMan ${pom.version} Installer

PubMan is a Web-based solution for the management of publication data for research organisations. It is based on a Service-oriented Architecture (escidoc.org) and comprises several Web-Services.

eSciDoc is a complex software stack, consisting of various software components. This installer will guide you step by step through the installation process. However, it may be necessary to install and configure a runtime environment prior to running this installer. The runtime environment consists of two components:

PubMan / eSciDoc Prerequisites:

  • Java 1.6 JRE
  • PostgreSQL Database 9.x

Initial Dataset

If you have selected this package option, the installer creates an initial dataset on the Core Service instance. This initial dataset consists of the following sample data objects:

  • 1 content model for publications
  • 1 content model for batch import tasks
  • 2 users: 'pubman_depositor' and 'pubman_moderator'
  • 2 organizational units: 'PubMan Default Organization' and 'External Organization'
  • 1 context: 'PubMan Default Context' with a simple workflow
