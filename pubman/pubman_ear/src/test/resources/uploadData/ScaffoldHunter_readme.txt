Installation of Scaffold Hunter
===============================

1.) copy the corresponding file "Scaffoldhunter.jar" to any folder on your hard drive. We recommend creating the folder "C:\Program Files\ScaffoldHunter" and use it as destination.
2.) copy the file "run_ScaffoldHunter.bat" to this folder

The installation of Scaffold Hunter is now complete.

Please note that Scaffold Hunter requires several other software to work. The following programs need to be installed in the version(s) stated below:

- Java (version 1.5 or later) which can be downloaded from http://www.java.com/en/download/inc/windows_upgrade_xpi.jsp
- MySQL Community Server versions 5.1 or 5.4 that can be downloaded from http://dev.mysql.com/downloads/mysql/5.1.html#downloads
- MySQL GuiTools 1.0 or later from http://dev.mysql.com/downloads/gui-tools/5.0.html

For the installation of the MySQL community server and the Gui tools, please refer to the corresponding manual available at http://edoc.mpg.de/display.epl?mode=doc&id=429252 and the MySQL documentation.



Running Scaffold Hunter
=======================
Start the Scaffold Hunter by double clicking on the ScaffoldHunter.bat file.
If the program does not start, your will have to add the Path to your Java Runtime Environment at the beginning of the first line in the bat file.
Alternatively, you can also right click on the scaffoldHunter.jar file, choose "run as" and select "Java (TM) SE Platform Binary". This will execute Scaffold Hunter with the standard Java settings.

Please note that using the bat file is recommended since it grants 1024 MBytes of Memory to Scaffold Hunter. Additional information of running Scaffold Hunter can be found in the Scaffold Hunter manual.
